# 발전소 지도 (power-plant-map)

전국 발전소(운영중·건설예정·폐지예정)를 지도에서 보여주는 웹서비스.
기획안: [발전소지도-서비스-기획안.md](../발전소지도-서비스-기획안.md)

## 구조

```
power-plant-map/
├─ pipeline/            데이터 파이프라인 (Python)
│  ├─ raw/              원천 데이터 (EPSIS, WRI, 시군구 경계)
│  ├─ curated/          11차 전기본 큐레이션 (plan11.json)
│  └─ build.py          조인·좌표 매칭 → web/public/data/plants.json 생성
└─ web/                 프론트엔드 (Vite + React + TS)
   └─ src/map/          지도 어댑터 (네이버 ←→ Leaflet 자동 전환)
```

## 실행

```bash
# 데이터 재생성 (원천 갱신 시에만 필요 — plants.json은 이미 생성돼 있음)
cd pipeline && python build.py

# 웹 실행
cd web && npm install && npm run dev   # http://localhost:5174
```

## 네이버 지도 활성화 (사용자 직접 발급 필요)

키가 없으면 개발용 OSM(Leaflet) 지도로 동작합니다. 네이버 지도로 전환하려면:

1. [네이버클라우드플랫폼](https://www.ncloud.com) 가입 + 결제수단 등록
2. 콘솔 → **Application Services → Maps** 이용 신청 (신규 Maps 상품, 2025.7~)
3. Application 등록 → **Web Dynamic Map** 선택, 서비스 URL에 `http://localhost:5174` 추가 (배포 시 실제 도메인도 추가)
4. 발급된 **Client ID(ncpKeyId)** 를 `web/.env` 에 설정:
   ```
   VITE_NCP_KEY_ID=발급받은키
   ```
5. dev 서버 재시작 → 자동으로 네이버 지도로 전환

무료 한도: Web Dynamic Map 월 600만 로드 (2026-07 기준, VAT 별도 초과 0.1원/건).

## 주요 기능

- **네이버부동산식 줌 클러스터링**: 줌 7 이하 시도 묶음(17개) → 줌 8~9 시군구 묶음 → 줌 10+ 개별 핀. 클러스터 클릭 시 단계별 확대
- **직관적 핀**: 연료원별 아이콘(⚛️원자력 🪨석탄 🔥LNG 🛢️유류 💧수력 🔁양수) + 용량 + 폐지/건설 배지
- **호버 미리보기 카드**: 마우스오버 시 이름·연료·용량·회사·상태·폐지 일정 요약
- **필터**: 연료원 9종 × 회사(한수원·발전5사·지역난방·수자원·민간·건설계획) × 상태 × 소규모 토글
- **호기별 상세**: 설비용량·준공연월·주기기 제작사(보일러·터빈·발전기, 열에 마우스오버 시 구분 표시)·폐지 예정연도
- **폐지↔대체 연결선**: 석탄 발전소 선택 시 11차 전기본 대체건설 부지로 점선 표시, 지연 현황 주석 + 호기 테이블에 대체호기 인라인 표기
- **발전량·이용률**: EPSIS 연간 발전기별 발전량 조인 — 발전소·호기 단위 이용률 표시 (2024년 기준)
- **가동 연차**: 최초 호기 준공연도 기준 "n년차" 표시
- **뉴스**: 사이드바 전력산업 주요 뉴스 + 발전소 상세의 관련 뉴스(태그 연동), 월간 정기 작업이 큐레이션
- **풍력**: 에너지공단 풍력기 위치정보(2025-12 기준) 118개 단지 — 단지 단위 집계, 시군구 근사 좌표. 사업자 정보는 원천에 없어 미표기
- **해양(조류)**: 진도 울돌목 조류발전 시험시설 1곳(한국해양과학기술원) — EPSIS '해양 에너지' 분류
- 태양광: 전수 통합 예정(지자체 136개 표준데이터 병합 필요 — 별도 작업)

- **🏠 우리 동네 혜택 판정기**: 시도→시군구→읍면동 선택 → 발주법 주변지역(발전기 반경 5km가 걸치는 읍면동 전체) 해당 여부 판정 + 혜택 안내(전기요금 보조·장학금·특별지원) + 연간 기본지원사업비 추정(발전량×단가). 판정 데이터는 `pipeline/benefits.py`가 생성(전국 1,320개 읍면동 해당). **안내용 추정이며 법적 판정 아님** — 좌표 근사·2013 행정경계·10MW 미만 제외 한계 명시. 단가는 시행령 별표2(2017.12.26 개정) **원문 확인 완료**: 원자력 0.25/유연탄 0.18/무연탄 0.3/유류 0.15/LNG 0.1/양수·수력·조력 0.2/신재생 0.1원/kWh + 양수 50만원/MW·수력 500만원/MW 설비용량 단가

## 알려진 한계 (기획안 §6 참조)

- 좌표 정밀도: 대형 발전소는 정확 좌표(용량 기준 68%), 나머지는 시군구 중심점 근사(상세 패널에 표기)
- 소수력·도서 내연 8곳은 원천 주소의 옛 지명 탓에 좌표 없음 (전부 15MW 이하)
- 폐지→대체 연결선은 대체 부지가 확정된 20건만 표시 (10건은 부지 미정)
- LNG 대체건설 준공 시기는 계획 기준이며 실제 다수 지연 중 (상세 패널 note 참조)
- 근무인원 데이터는 미포함 (사업소 단위 공개 데이터 부재 — 후속 조사 필요)
- 주기기 **기종·모델명**은 공개 데이터에 없어 제작사까지만 제공 (EPSIS는 제작사만 공개)
- 건설예정 설비 중 일부(구미·공주·용인 등)는 정확한 부지 좌표가 공개되지 않아 시군구 중심점 근사 + 용량은 대체 대상 호기 합계 기준 추정치로 표시(패널에 명시). 기존 부지 내 건설이 확인된 보령·하동 신복합은 정밀 좌표 적용
